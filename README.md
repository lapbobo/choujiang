# 幸运大抽奖 WebApp (V2.0)

一款专为线下大屏投屏场景设计的轻量级抽奖 Web 网页。

## 功能特性

### 核心功能
- ✅ 抽奖号码范围自定义（默认 1-200）
- ✅ 奖项配置自定义（默认：一等奖 1 个、二等奖 2 个、三等奖 3 个、幸运奖 5 个）
- ✅ 键盘快捷键控制（空格键开始/停止抽奖）
- ✅ 去重逻辑（已中奖号码自动移出奖池）
- ✅ 配置持久化（使用 LocalStorage 自动保存）
- ✅ 导出中奖名单为图片

### 视觉体验
- 🎨 糖果色暖色系设计
- 🎊 中奖彩带动画效果
- 🌟 动态数字滚动动画
- ✨ 发光和缩放动画
- 📱 响应式设计，支持横屏和竖屏

### 音效设计
- 🔊 按钮点击音效
- 🚀 开始抽奖音效
- ⚡ 数字滚动音效
- 🎉 中奖礼炮/欢呼声
- 🎊 完成抽奖音效

### 其他特性
- 🖥️ 大屏适配（支持全屏模式）
- ⚙️ 浮动弹窗设置界面
- 🔄 一键重置为默认配置
- 💾 自动保存抽奖结果

## 使用方法

### 快速开始

1. 直接用浏览器打开 `index.html` 文件
2. 点击"开始抽奖"按钮或按"空格键"开始抽奖
3. 再次点击或按"空格键"停止抽奖，显示中奖号码
4. 重复上述步骤，直到所有奖项抽完

### 自定义设置

1. 点击右上角的"抽奖设置"按钮
2. 在弹窗中修改抽奖号码范围
3. 添加、删除或修改奖项配置
4. 点击"保存设置"应用更改

### 导出结果

1. 抽奖结束后，点击"导出图片"按钮
2. 图片将自动下载到本地

### 快捷键

| 按键 | 功能 |
|------|------|
| 空格键 | 开始/停止抽奖 |
